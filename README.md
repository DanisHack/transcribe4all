# audio-transcription-service

## Setting up

Go get the project

```
$ go get github.com/hack4impact/audio-transcription-service
$ cd $GOPATH/src/github.com/hack4impact/audio-transcription-service
```

Install the dependencies

```
godep restore
```

## Running the app

```
go run main.go
```
